# 王阳明心学

核心：1.心即理，逻辑起点	2.知行合一，实践品格	3.心外无物

总结：无善无恶是心之体，有善有恶是意之动，知善知恶是良知，为善去恶是格物

# 本体与主体性

**致良知：**

每个个体存在性的真实表达是要通过我们自觉的道德践履来实现的，而只有这样子，我们的整个生命过程才不仅仅是合乎他的自身本然状态的，同时使合乎天道的。

**良知：**三个层面

1. 存在之本源

2. 德行之本源

3. 理性之本源

# 致良知第一层意思

呼唤自我的良知：只有一条途径--反思

反省：把日常生活中的各种各样的观点、意识、行为（所想、所说、所作）直接的放到心灵面前进行自我的关照

只有经由自我反思，我们的主体性才得以建立起来

合乎良知的是诚，不合良知的是不诚

# 致良知第二层意思

致吾心之良知于事事物物，就是明明德于天下

每一个人的生活世界，就是通过良知的自我表达而被建立起来的
