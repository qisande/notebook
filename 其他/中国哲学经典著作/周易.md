# 乾卦

- 天行健，君子以自强不息

# 坤卦

> **卦辞：**  *元亨。利牝马之贞。君子有攸往，先迷后得主，利。西南得朋，东北丧朋。安贞吉。*

- 地势坤，君子以厚德载物

- 柔顺正固；后发优势；明确位置，找到同类。 

- 勿以恶小而为之，勿以善小而不为
